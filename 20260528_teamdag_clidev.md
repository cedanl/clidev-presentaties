# Ronde 1 — Verkenning

<div class="grid grid-cols-2 gap-6" style="font-size: 0.9rem; line-height: 1.7; margin-top: 1rem;">
<div>

**Doel:** Vind een goede dataset voor synthese

- Gebruik **[Onderwijs Data Chat](https://github.com/cedanl/onderwijsdata-chat)** om een dataset te vinden
- Welke dataset kies je en voor welke analyse zou die bruikbaar zijn?

</div>
<div>

**Feedback op Onderwijs Data Chat:**
- Wat werkt goed, wat niet?
- Wat wilde je doen maar kon niet?
- Wat zou je anders willen?
- Wat zou je aanpasbaar willen?

**Bonus**
- Wat denk je van gebruik van deze tool?
- Wat denk je van eigen projecten nu je dit hebt gezien?

</div>
</div>

# Ronde 2 — Synthetiseren (45 min)

Maak een synthetische versie van je dataset en toon de kwaliteit aan.

- **Wat** is synthetische data?
- **Wat bepaalt** of iets goed is gesynthetiseerd?
- **Wat is je definitie** en validatie van kwaliteit?

Code delen!
